# Presentation Website

Open `index.html` in a web browser. The website works offline.

## Controls
- Use the tabs at the top or the Previous / Next buttons.
- The browser Back and Forward buttons also change sections.
- Left and Right arrow keys move between sections.
- “Presentation mode” opens the page in fullscreen when the browser supports it.
- Diagram buttons open the SVG models in a zoomable full-screen window.

## Presentation order
Overview → Entities → Theoretical model → Conceptual model → TEI/XML → RDF/LOD → Validation → Presentation guide.
